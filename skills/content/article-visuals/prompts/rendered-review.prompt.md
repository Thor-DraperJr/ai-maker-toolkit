# Rendered Review Prompt

```text
Review this rendered article visual or HTML presentation as a teaching artifact.

Check:
- Does the article/source still make sense without the visual?
- Does the visual teach one clear idea?
- Does motion explain sequence, state, flow, contrast, or emphasis?
- Does any moving object appear before the path/state it depends on exists?
- Are labels readable and unclipped?
- Does the presentation viewport fit?
- Do keyboard navigation, counter/title updates, speaker cues, and exit behavior work?
- Are generated screenshots and audit artifacts ignored or cleaned?

Return blockers first, then required revisions.
```
