# Visual Brief Prompt

```text
Create a visual brief for this article section or teaching moment.

Source section:
<paste or reference section>

The visual should teach:
<one idea>

Decide whether the visual should be static, animated, or interactive.
If animated, explain what moves, when it starts, what remains static, and how reduced motion should work.
Return labels, layout, states, motion timing, viewport checks, and screenshot review criteria.
```
