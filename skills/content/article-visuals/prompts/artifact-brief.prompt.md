# Artifact Brief Prompt

```text
Create an artifact brief for this source material.

Source:
<paste or reference source>

Audience:
<target audience>

Desired outcome:
<what the reader or audience should be able to do>

Classify the artifact as article-only, visual-heavy, presentation-ready, or standalone HTML presentation.
Name the visual needs, animation needs, optional present-mode decision, proof gates, and repo hygiene risks.
Return the brief in Markdown.
```
